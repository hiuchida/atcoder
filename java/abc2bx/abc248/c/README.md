# Main
n,m,kを読み込む。
int[][] dpを用意し、iを1からmまでループし、dp[1][i]=1で初期化する。
hを2からnまでループし、iを1からkまでループし、jを1からmまでループする。
i+j>kを範囲外ならスキップする。
dp[h][i+j]にdp[h-1][i]を加える。
ans=0に初期化し、dp[n][i]をansに加える。
ansを出力する。

# Main\_fix
Mainを元に、dp[1][i]=1の初期化もh=1で振り分ける。
dp\[0\]\[0\]=1で初期化する。
iを0からkまでループする。（k+1>kのため、厳密にはk-1までのループでよい）
