# Main
2020年10月に提出して正解。
5^bは、必ず1の位が5となる。
3^aは、3,9,27,81,243と3,9,7,1とサイクルする。
このため、nの1の位が、8,4,2,6以外は-1を出力する。
a=1から4、b=1からループしてnと一致する組み合わせを出力する。
