このコードは、全てのあり得る油田の配置（全プール）を列挙し、ランダムな座標を占った結果から各配置の**対数尤度**を更新して、最も尤度が高い配置を特定する手法を実装しています。

*   **対数尤度の利用**: 確率（1以下の値）を掛け合わせ続けると精度が足りなくなるため、`Math.log` を用いた加算形式で計算しています。
*   **誤差関数(erf)**: C++の標準ライブラリにはありますが、Javaの `Math` クラスにはないため、数値的な近似式を実装に含めています。
*   **累積分布関数(CDF)**: 占い結果 $r$ が得られる確率は、$\text{CDF}(r+0.5) - \text{CDF}(r-0.5)$ で求めます。ただし $r=0$ の場合のみ、$-\infty$ から $0.5$ までの範囲を計算します。
*   **全プールの限界**: このコードはソース内の「全プール」手法に基づいているため、$M=2$ 程度までは動作しますが、$M \ge 3$ では配置候補数が爆発し、メモリ不足やTLE（実行時間制限超過）になります。その場合はソースで紹介されている「数を絞ったランダムなプール」や「焼きなまし法」への拡張が必要です。
