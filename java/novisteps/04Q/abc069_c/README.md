# Main02
2 2は消えるが、2 2 2と奇数は1個残ると思っていた。2個以上は0個にする。
Main01を元に、

n2%=2を入れ、
を
n2>1の場合、n2=0を入れる。
に変更。
WA2個。

# Main03

Main02を元に、
n4*2>=n2+n1の場合ok、それ以外はng。
を
n4+1>=n2+n1の場合ok、それ以外はng。

# Main
Main02で、1 2 2のような場合、2 2部分は消えるが、1 2は残ることを勘違いしていた。
よって、2が1個でも100個でも、1が1個として扱う。

n2>1の場合、n2=0を入れる。
を
n2>1の場合、n2=1を入れる。
に変更。
