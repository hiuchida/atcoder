# Main01
問題文の「その行の左から i 個のマスは黒、それ以外は白で塗られている。」のそれ以外が何以外か
腑に落ちていなくて、仕様レビューがないと厳しい。

黒と白と別々にmapで管理したので、とりあえず白より右下に黒が存在したらNGという判定で作ったらWA19個。
